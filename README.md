# Speedyclearprint Website Redesign

A modern, responsive single-page website for **Speedyclearprint**, a Johannesburg-based print shop specializing in litho printing, digital printing, promotional products, apparel branding, and large-format printing.

The project was built using **React**, **JavaScript**, and **Tailwind CSS**, with a focus on improving the company's online presence through a clean editorial design, responsive layouts, and improved user experience.

---

## Live Features

- Modern editorial-inspired user interface
- Fully responsive design (mobile, tablet, desktop)
- Sticky navigation with smooth scrolling
- Hero section with company highlights
- About section
- Services section with authentic company images
- Complete capabilities list
- Five-Step Print Process timeline
- Company values section
- Client showcase
- Contact section
- Functional quote form (mailto)
- Click-to-call phone links
- Floating WhatsApp contact button
- Interactive hover animations
- Paper-inspired colour palette

## Design Improvements

Compared to the original Speedyclearprint website, this project introduces several enhancements:

- Editorial-inspired modern design
- Improved typography using Fraunces and Inter Tight
- Better spacing and visual hierarchy
- Responsive navigation with sticky header
- Cleaner service presentation using image cards
- Authentic company photography
- Interactive hover animations
- Floating WhatsApp support
- Simplified customer contact experience
- Improved accessibility and responsiveness
